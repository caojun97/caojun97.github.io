Hugo+Github个人博客的搭建

在学习过程中，不可避免的要记录学习笔记。学生时代，普遍使用纸质笔记本记录笔记，但纸质笔记有一个致命的缺陷–遗失不可恢复。参加工作后，逐渐将纸质笔记转移到电子笔记，电子笔记可以随时记录、随时查询等优点。

市面上电子笔记app有很多，例如OneNote、印象笔记、Notion。目前我也是使用Notion，当然OneNote和印象笔记也简单的使用过。这类笔记app模板很多、上手简单，但也有几个缺点：

  1. 笔记资料存储在别人服务器上（服务器关闭服务，无法找回笔记）
  2. 笔记资料上传和下载有限制（充值可以解除限制）
  3. 应用服务器部署在国外，访问电子笔记受网络限制（访问卡顿）

搭建个人博客记录学习笔记也是必须的，开源搭建博客应用有很多，这里使用了UP：CoodSheep 推荐的Hugo。

1、Hugo的介绍和安装

Hugo是一款开源的、受欢迎的 静态站点生成器。可以根据用户自定义主体，搭建个性化个人Blog。

1-1 下载和安装Hugo

本地系统环境：Windows 10 专业版 64位

Git版本：git version 2.34.1.windows.1

Hugo下载地址：Hugo_0.94.2_Windows-64bit.zip

  • 下载Hugo压缩包

  • 解压Hugo压缩包

  • 设置hugo.exe环境变量

    在cmd.exe窗口中输入命令：

    hugo version

​

1-2 创建Blog站点

  • 创建Blog站点

    在cmd.exe窗口中输入命令：

    cd /d d:\document
hugo new site myblog

  • 下载Hugo主题

    Hugo主题下载路径：[Hugo主题](Complete List | Hugo Themes (gohugo.io))

    个人博客选择的主题：m10c

    cd /d d:\document\myblog\
git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c

    在config.toml文件中新增一行内容

    theme = "m10c"

  • 创建博客文件

    cd /d d:\document\myblog
hugo new post\Hugo+Github个人博客.md

    利用Typora应用编写markdown格式的文章

1-3 启动Hugo服务

文章编写完毕，利用Hugo发布文章，即可通过浏览器访问文章

cd /d d:\document\myblog
hugo server -t m10c --buildDrafts

1-4 浏览器访问服务

在Edge浏览器地址栏输入:http://127.0.0.1:1313，验证文章是否发布成功

2、 Github同步本地个人博客

2-1 Github创建新仓库

![image-20220315190021269](D:\document\myblog\static\img\屏幕截图 2022-03-15 200425.png)

注意：图中Repository name的填写格式，例如 caojun97.github.io

2-2 个人博客部署到远端服务器

  • 主题和文章格式转换

    cd /d d:\document\myblog
hugo --theme=m10c --baseUrl="https://caojun97.github.io/" --buildDrafts

    命令执行后，在d:\document\myblog\public\会将博客转换成HTML和CSS格式

  • 建立本地git仓库

    cd /d d:\document\myblog\public\
git init
git add .

  • 关联远端github仓库

    cd /d d:\document\myblog\public\
git commit -m "我的hugo博客第一次提交"
git remote add origin https://github.com/caojun97/caojun97.github.io.git

  • 推送远端github仓库

    cd /d d:\document\myblog\public\
git push -u origin master

  • github仓库验证

  • github仓库链接访问博客

    在Edge浏览器地址栏输入:https://caojun97.github.io/，验证文章是否发布成功
